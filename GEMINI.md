# Gempill

**Goal:** A Material 3 Expressive meds tracker.

**Features:**

*   Home page with today's schedule (actionable status buttons)
*   Records page with calendar view of daily adherence %
*   List of active meds
*   FAB for adding new meds (schedule, dosage, times)
*   Clickable med detail pages (edit, log history)
*   Local/native reminders
* Project strictly in line with Material 3 Expressive design language and guidelines

**Architecture:**

*   Must use maximum modularity, separating UI elements into distinct, reusable Components following the Container/Presentational pattern.

**Tech Stack:**

*   React Native
*   TypeScript
*   Bare React Native CLI with Expo Modules
*   Styling must primarily use `StyleSheet.create`
